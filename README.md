# 🌸 Shen Arte Floral

**Shen Arte Floral** es una plataforma web desarrollada para una florería online enfocada en entregar detalles que emocionan, con sede en la hermosa ciudad de Puerto Montt. Nos especializamos en crear arreglos únicos con una gran variedad de flores frescas para sorprender en cualquier ocasión.

## 📌 Características Principales

* **Diseño Atractivo y Responsivo**: Interfaz gráfica moderna, intuitiva y adaptable a cualquier dispositivo (móviles, tablets y escritorio).
* **Navegación Fluida**: Secciones interconectadas para descubrir Servicios, Productos y un apartado de Contacto.
* **Carrusel de Imágenes**: Exhibición dinámica de los arreglos destacados de la florería en la página principal.
* **Experiencia de Usuario (UX)**: Diseño orientado a transmitir la belleza del arte floral, utilizando tipografías, colores y componentes visuales atractivos.

## 💻 Tecnologías Utilizadas

* **HTML5**: Estructuración semántica y accesible del contenido web.
* **CSS3**: Estilos personalizados (`css/style.css`) para darle una identidad única a la marca.
* **Bootstrap 5.3**: Framework utilizado para el sistema de grillas responsivo y componentes interactivos.
* **JavaScript**: Lógica de interactividad para componentes visuales proporcionados por Bootstrap.

## 📂 Estructura del Proyecto

El proyecto está organizado de la siguiente manera para mantener un código limpio y escalable:

```text
Shen-Arte-Floral/
├── css/
│   └── style.css            # Hoja de estilos personalizados
├── html/
│   ├── Servicios.html       # Página detallando los servicios ofrecidos
│   ├── Productos.html       # Catálogo de arreglos florales
│   └── Contacto.html        # Formulario e información de contacto
├── img/                     # Recursos gráficos e imágenes de la florería
├── js/                      # Scripts de comportamiento interactivo
└── index.html               # Página principal (Landing Page)
```

## 🚀 Cómo Visualizar el Proyecto

Al ser un proyecto Front-end estático, su ejecución es muy sencilla y no requiere la instalación de servidores locales:

1. Clona este repositorio en tu máquina local o descarga el archivo `.zip`.
2. Navega hasta la carpeta del proyecto.
3. Haz doble clic en el archivo `index.html` o arrástralo hacia tu navegador web preferido (Google Chrome, Mozilla Firefox, Safari, Edge, etc.).
